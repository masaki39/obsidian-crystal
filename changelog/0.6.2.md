## What's Changed

### Removed

- **Quick Add 機能を廃止**: `Quick add: Add task to todo list` コマンドと、設定の **Quick Add** セクション（ToDo File Name / Inbox Name）を削除した。

### Changed

- **「今日のデイリーノートへタスク追記」を Daily Notes へ統合**: `Daily: Add task to daily note` コマンドはそのまま利用できる。実装を `DailyNotesManager` に移し、専用クラス（`quick-add-commands.ts`）と重複していたデイリーノート探索ヘルパーを整理した。
